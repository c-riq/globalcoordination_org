globalcoordination.org

We aim to enable the enforcement of international rules via effective digitial global coordination.

The coordination platform stated.network aims to:

  • provide an open source and decentralised 'social media' platform for governments
  • enable faster negotiation and finalization of agreements among nations
  • provide increased transparency for the public
  • improve the reaction time of collective rule enforcement to violations

Using a simple analogy, it is like a combination of twitter and docusign built for the governments, foreign ministries and international organisations.

Roadmap

  • Help governments and organisations to coordinate more effectively on a global scale using stated.network.
  • Enable individuals globally to express their preferences in real time using ip-vote.com. Where every IP address and phone number on earth can be used to vote once for each poll.
